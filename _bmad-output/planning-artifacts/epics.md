# Épicas de Downstroke

## Epic 1: MVP instalable y verificable

Downstroke debe pasar de baseline documental a framework modular capaz de instalar disciplina de proyecto sin sobrescribir trabajo del usuario.

### Story 1.1: Preservar baseline e importar fuentes

Como autor del framework, necesito conservar el baseline y versionar las guías fuente para extraer módulos sin perder conocimiento ni alterar archivos originales.

### Story 1.2: Crear fundación del monorepo

Como maintainer, necesito una estructura mínima de workspaces para desarrollar core, módulos y CLI con límites claros y checks compartidos.

### Story 1.3: Inicializar y diagnosticar preset lite

Como developer, necesito ejecutar `downstroke init --preset lite` y `downstroke doctor` para instalar el baseline mínimo de forma segura y comprobar su salud.
