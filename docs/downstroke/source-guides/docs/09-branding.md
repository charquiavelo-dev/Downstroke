# Downstroke Branding

## Name

```txt
Downstroke
```

## Meaning

In guitar, a downstroke is a controlled downward pick attack. In metal and thrash, repeated downstrokes create force, discipline and precision.

In software, Downstroke means:

- disciplined execution;
- strong rhythm;
- clean gates;
- no sloppy AI output;
- project structure that survives pressure.

## Tagline Options

```txt
Disciplined AI-assisted delivery.
```

```txt
Hard rhythm for clean software.
```

```txt
Small modules. Strong gates. Real delivery.
```

```txt
AI delivery with rhythm, grit, and gates.
```

Recommended:

```txt
Small modules. Strong gates. Real delivery.
```

## Voice

Downstroke should sound:

- direct;
- technical;
- confident;
- slightly aggressive;
- never childish;
- never corporate fluff.

## Visual Direction

Logo ideas:

- black guitar pick shaped like a shield;
- downward slash/cut inside the pick;
- subtle gate/bar motif;
- sharp but not unreadable;
- black on white first;
- optional red accent later.

Avoid:

- skull-heavy parody logo;
- unreadable death metal font for UI;
- too much texture;
- childish rock icons;
- generic code brackets as the main logo.

## Package Naming

Use clean package names:

```txt
@downstroke/core
@downstroke/spec
@downstroke/agents
@downstroke/gates
@downstroke/presets
```

Avoid:

```txt
@downstroke/brutalizer
@downstroke/killchain
@downstroke/hellgate
```

Brand can be metal. APIs should stay readable.
