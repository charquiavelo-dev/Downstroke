# Story 018: Project Foundation MD

## Goal

Como developer o cliente tecnico, quiero describir brevemente lo que quiero construir para que Downstroke genere un MD inicial con filosofia, idea, primeros pasos y ruta para profundizar.

## Tasks

- Crear comando `downstroke project foundation`.
- Pedir descripcion corta del producto.
- Preguntar usuario objetivo, problema, stack existente y velocidad esperada.
- Usar LLM activo si esta disponible.
- Crear `docs/downstroke/PROJECT_FOUNDATION.md`.
- No escribir archivos en la raiz.
- Incluir Downstroke Method y Breakdown Stack.
- Incluir primeros pasos, riesgos y decisiones pendientes.

## Acceptance

- El documento se crea dentro de `docs/downstroke/`.
- El documento no aparece en la raiz.
- El contenido explica filosofia del proyecto e idea inicial.
- El contenido guia los primeros pasos de desarrollo.
- El contenido explica como profundizar despues.
- El tono comunica una herramienta robusta para developers reales.
