# Story 001: Preserve Legacy Boilerplate

## Goal

As the framework author, I want the current boilerplate preserved as a legacy example so Downstroke can evolve without losing the working baseline.

## Tasks

- Create `examples/legacy-boilerplate/`.
- Copy current boilerplate files into that folder.
- Add README explaining it is the source baseline.
- Tag initial repo state as `v0.0.0-boilerplate`.

## Acceptance

- Legacy boilerplate exists unchanged.
- Downstroke docs say migration is additive.
- No original files are deleted or rewritten.
